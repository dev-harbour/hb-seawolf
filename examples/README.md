# hb-seawolf

## Installing on Windows

- MSYS2

   ```
   pacman -S mingw-w64-x86_64-glfw
   ```
   To test it, type:

   ```
      $ cd examples/main
      $ hbmk2 main.prg
   ```

## Installing on Linux

- Debian

   ```
   sudo apt-get update
   sudo apt-get install libglfw3-dev
   ```
   To test it, type:

   ```
      $ cd examples/main
      $ hbmk2 main.prg
   ```
